# Pong
Arcade game Pong [[Wikipedia](https://en.wikipedia.org/wiki/Pong)]
<br>
<br>

## Requirements

- Python 3.5.4

## Default keyboard controls

Left player:<br>
Up - `d`<br>
Down - `c`

Right player:<br>
Up - `k`<br>
Down - `m`
<br>

Pause or resume game - `Space bar`
